# MavenAppStarter
Starter template for maven projects

## Available API routes
* Book routes
* video routes

### Book routes
#### {serverName}/api/v1/
* addBook
CREATE a book

* books
GET all books, returns JSONData

* book/{id}
GET book with a given id

* bookName/{book}
GET book with unique name

* update/{id}
UPDATE a book

* delete/{id}
DELETE a book

### Video routes
#### {serverName}/api/v1/
* addVideo
* video
* video/{id}
* videoName/{book}
* update/{id}
* delete/{id}
